À propos

Passionné par l'histoire et désormais engagé dans le développement informatique,
je poursuis actuellement un BTS SIO (Services Informatiques aux Organisations).

Cette formation me permet d'acquérir des compétences techniques avancées en programmation
et gestion des systèmes d'information.

Le GRETA, organisme de formation continue, est un partenaire clé dans ce parcours,
offrant des ressources et un soutien précieux pour développer mes compétences professionnelles.

Avec cette combinaison unique, je suis prêt à aborder chaque projet avec créativité et expertise technique.

Mes Compétences

CSS

Le CSS ou Cascading Style Sheets, est utilisé pour définir l'apparence et le formatage des pages web.
Grâce au CSS, on peut contrôler divers aspects tels que la mise en page
les couleurs, les polices et les espacements, tout en assurant une cohérence de style sur l'ensemble du site.

HTML

HTML, ou HyperText Markup Language, est le langage de balisage standard utilisé pour créer et structurer le contenu des pages web.
Il permet de définir la structure des documents en utilisant des balises pour organiser le texte, les images et les autres éléments de contenu.
Une bonne maîtrise du HTML est essentielle pour tout développeur web, car il constitue la base sur laquelle
les feuilles de style CSS et les scripts JavaScript sont appliqués.

Javascript

JavaScript est essentiel pour les fonctionnalités interactives telles que les formulaires dynamiques, les animations, et les mises à jour de contenu en direct.
Grâce à sa compatibilité avec divers environnements de développement, il joue un rôle clé dans le développement web moderne
facilitant la création d'expériences utilisateur riches et engageantes.
